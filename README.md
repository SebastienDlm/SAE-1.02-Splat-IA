# Projet Universitaire - Python : SAE 1.02  
## Projet Python 2 - Splat IUT'O

Un projet universitaire visant à créer une IA capable de jouer à un jeu avec des choix programmés selon les situations.

Nous avions un projet de base qui ne se trouve pas dans ce dépôt, car il ne m’appartient pas.

Splat IUT’O est un jeu développé par le département informatique de l’Institut Universitaire d’Orléans.  
Le but du jeu est de colorier le plus de cases possible dans un temps imparti. Il se joue à 4 joueurs, chacun pour soi.

En tant qu’étudiants, notre objectif était de créer une IA capable de jouer seule au jeu avec des actions programmées à l’avance. Dans notre équipe, le bot s’appelle *Les Steamers* (*Steamy* pour les intimes).

Le code que nous avons développé à quatre se situe dans le fichier `cerveau.py`, ainsi que dans la fonction `mon_IA`, qui se trouve dans `client_joueur.py`.  
Avant de nous occuper du développement de l’IA, nous avions quelques fonctions à implémenter dans les fichiers `joueur.py` et `plateau.py`.

Merci de votre intérêt pour ce projet.
