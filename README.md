# Projet_SplatIUTO
Un projet universitaire visant à créer une IA qui joue à un jeu avec des choix programmés selon les situations.

Nous avions un projet de base qui ne se trouve pas dans ce dépôt car il ne nous appartient pas.

Splat IUT'O est un jeu développé par le département informatique de l'Institut Universitaire d'Orléans.
Le but du jeu est de colorier le plus de case dans un temps imparti. 4 joueurs, chacun pour soi.

En tant qu'étudiant, notre objectif était de créer une IA pouvant jouer toute seul au jeu avec des action programmée à l'avance.

Le code que nous avons développé à 4 se situe dans le fichier cerveau.py et la fonction mon_IA qui se trouve dans client_joueur.py.
Avant de nous occuper de développer l'IA, nous avions quelques fonctions à implémenté dans les fichiers joueur.py et plateau.py.


Merci de votre intérêt pour ce projet.
